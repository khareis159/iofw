Este diretório contém os arquivos fundamentais para a execução da 
Faculdade Simbiótica HyperWeb3. 

1. ESTRUTURA DE ARQUIVOS:
   /main
   ├── index.html     (Interface Principal e Motor do Dashboard)
   ├── manual.html    (Códice do Iniciado - Manual Acadêmico)
   ├── leiame.txt     (Este documento)
   └── manifesto.txt  (Descrição da Engenharia e Filosofia)

2. INSTRUÇÕES DE INSTALAÇÃO:
   A) Carregar os arquivos 'index.html' e 'manual.html' no repositório GitHub.
   B) No Firebase, configurar as 'Database Rules' para permitir leitura/escrita.
   C) No 'index.html', inserir a chave API da Groq na linha indicada (CHAVE_IA).
   D) Ativar o GitHub Pages nas configurações do repositório.

3. USO DO SISTEMA:
   - O sistema solicitará um Nome de Pioneiro no primeiro acesso.
   - O progresso (Rigor, Imaginação e Conversas) é salvo automaticamente.
   - Utilize o botão "Exportar Legado" para gerar o comprovante de estudo.
